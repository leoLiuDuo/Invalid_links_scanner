it is a simple python script to help you scan your website whether there are anyinvalid or break links.
python version   3.6

require package:
bs4(you can use: pip install bs4)

input:
  main address example: http://www.rishiqing.com
  domain_key example: rishiqing

output:
  parent url: tell you which page the invalid url was found
  invalid url: the target witch is broken
